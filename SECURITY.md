# Security Policy

## Supported Versions

The following versions of **The Baker** are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

I take the security of this project seriously. If you believe you have found a security vulnerability, please do **not** report it via a public GitHub issue. Instead, please follow the steps below:

1. **Email me**: Send a detailed report to dev@indiedesert.com
2. **Details**: Include as much information as possible, including reproduction steps, impact, and any potential fix.
3. **Wait for acknowledgment**: I will acknowledge your report within 48-72 hours.
4. **Responsible Disclosure**: I ask that you do not publicly disclose the vulnerability until I have had a chance to investigate and release a fix.

Thank you for helping keep this project secure!
